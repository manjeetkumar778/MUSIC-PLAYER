

# 🎵 Music Player

A simple and responsive Music Player built using modern web technologies.  
This project allows users to play, pause, skip, and control audio tracks with an interactive UI.

## 🚀 Features
- ▶️ Play / Pause music
- ⏭️ Next and Previous track
- 🔊 Volume control
- 📊 Progress bar with seek option
- 🎨 Responsive and clean UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

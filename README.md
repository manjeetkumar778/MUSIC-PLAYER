# 🎵 Music Player Web App

A simple and responsive **Music Player** built using modern web technologies. This project allows users to play, pause, skip, and control audio tracks with an interactive UI.

---

## 🚀 Features

* ▶️ Play / Pause music
* ⏭️ Next / Previous track
* 🔊 Volume control
* 📊 Progress bar (seek functionality)
* 🎶 Playlist support
* 💻 Responsive design (Mobile + Desktop)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript 

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/manjeetkumar778/Music-Playerr.git
```

2. Navigate to the project folder:

```
cd music-player
```

3. Open `index.html` in your browser.

---

## 🎯 Usage

* Click the **Play button** to start music
* Use **Next/Previous** to switch songs
* Drag the **progress bar** to seek
* Adjust **volume slider**
